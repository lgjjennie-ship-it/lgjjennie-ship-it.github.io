## 一、分析背景与目标

### 1.1 背景
优云官网首页完成改版，需要通过数据验证改版效果，找到优化方向。

### 1.2 分析目标
1. **评估改版效果** - 新版首页的整体表现
2. **定位转化瓶颈** - 找到用户流失的关键节点
3. **指导后续优化** - 基于数据给出改进建议

### 1.3 核心问题
- 用户来了首页后，做了什么？
- 哪些区域最吸引用户？
- 从浏览到注册/使用，哪一步流失最多？

### 7.2 常见数据平台配置说明

#### 神策分析
```
漏斗分析 → 新建漏斗 → 添加步骤:
Step1: 事件=页面浏览, 筛选条件=页面路径包含"首页"
Step2: 事件=埋点事件名
...
窗口期: 建议设为同一会话 或 1天内
```

#### GrowingIO
```
产品分析 → 漏斗分析 → 创建漏斗:
添加步骤 → 选择事件 → 设置转化周期
```

#### 自研平台
```sql
-- 示例SQL逻辑
WITH funnel AS (
  SELECT 
    user_id,
    MAX(CASE WHEN event = 'page_view' AND page = 'home' THEN 1 ELSE 0 END) as step1,
    MAX(CASE WHEN event = 'nav_click_register' THEN 1 ELSE 0 END) as step2,
    MAX(CASE WHEN event = 'register_success' THEN 1 ELSE 0 END) as step3
  FROM events
  WHERE date BETWEEN '起始日期' AND '结束日期'
  GROUP BY user_id
)
SELECT 
  SUM(step1) as step1_count,
  SUM(step2) as step2_count,
  SUM(step3) as step3_count
FROM funnel;
```

## 八、附录

### 8.1 术语说明

| 术语 | 定义 |
|------|------|
| PV | Page View，页面浏览量，同一用户多次访问计多次 |
| UV | Unique Visitor，独立访客数，同一用户只计1次 |
| 跳出率 | 只浏览一个页面就离开的访问占比 |
| 转化率 | 完成目标行为的用户占总用户的比例 |
| 漏斗 | 用户完成目标的分步骤路径，用于分析各步流失 |
